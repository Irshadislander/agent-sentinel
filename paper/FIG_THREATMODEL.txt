Threats                                      Defenses
-------                                      --------

prompt injection / payload poisoning  --->   policy gating (pre-execution deny)

plugin override / unsafe extension     --->   allowlist isolation + plugin controls

silent failures / ambiguous errors     --->   structured errors + stable exit classes

missing forensic visibility            --->   trace completeness enforcement (TCR)
